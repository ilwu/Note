1.使用流程及注意事項

1.1.呼叫方式

API URL

https://{{API_DOMAIN}}/apiRequest
  • API_DOMAIN 當貴司申請文件填寫完後，我方會再把相關資料一起回覆
  • 請求和回應的 Content-Type 都為 application/json

HTTP Method

  • POST

請求資料格式





{
  "dc": "{{YOUR_DC}}",
  "x": "{{RSA_ENCRYPED_STRING}}"
}
  • 參閱API 功能項目決定要呼叫的功能
  • 依功能所需的參數轉為JSON字串後再進行加密
  • 用我方提供的public.pem進行RSA加密
  • 參數x的加密方法請以RSA PKCS1-OAEP進行加密，參考範例程式

回應資料格式

  • 成功回應
參數 格式 說明
status String(4) 0000
data Array / Object 回傳資料
requestId String 每次請求的獨立 Id 





{
    "status": "0000",
    "data": {},
    "requestId": "{{uniqueId}}",
}
  • 失敗回應
參數 格式 說明
status String(4) 0000 以外的代碼
message String 相關訊息
requestId String 每次請求的獨立 Id 





{
    "status": "9000",
    "message": "Error Messages",
    "requestId": "{{uniqueId}}",
}

    2.9.[61] 查詢遊戲詳細交易紀錄

    參數 格式 必填 說明

    belongsDC String N 代理域名，預設不需填入
    agentname String Y 代理帳號
    stime String Y 查詢區段起始時間

    YYYY-MM-DD HH:mm:ss
    etime String Y 查詢區段結束時間

    YYYY-MM-DD HH:mm:ss
    page String Y 第幾頁

    • 查詢單一代理在特定時間區段內的遊戲交易紀錄(依AddDate時間查詢)

    • 開始與結束時間中，ss(秒)的值必須為00

    • 無法查詢2 分鐘內或90 天以前的紀錄，每次最多回傳100 筆記錄，超過請分頁查詢

    • 以UTC+0 時區為基準

    static byte[] RSAEncrypt(byte[] DataToEncrypt, RSAParameters RSAKeyInfo, bool DoOAEPPadding)
    {
        byte[] encrypted = new byte[] {};

        using (RSACryptoServiceProvider RSA = new RSACryptoServiceProvider(2048))
        {
            RSA.ImportParameters(RSAKeyInfo);

            const int MAX_ENCRYPT_BLOCK = 100;
            int i = 0;
            int offSet = 0;
            List<byte> encryptList_ = new List<byte>(DataToEncrypt);

            while (encryptList_.Count - offSet > 0)
            {
                List<byte> sublist;
                if (encryptList_.Count - offSet > MAX_ENCRYPT_BLOCK)
                {
                    sublist = encryptList_.GetRange(offSet, MAX_ENCRYPT_BLOCK);
                }
                else
                {
                    sublist = encryptList_.GetRange(offSet, encryptList_.Count - offSet);
                }

                offSet = i * MAX_ENCRYPT_BLOCK;
